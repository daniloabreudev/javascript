# 🚀 Jornada JavaScript - Curso em Vídeo

Bem-vindo ao meu repositório de estudos dedicado ao aprendizado de **JavaScript**, acompanhando o excelente treinamento do professor Gustavo Guanabara no canal *Curso em Vídeo*. 

O objetivo deste repositório é documentar a minha evolução passo a passo, armazenar os códigos das aulas, registrar a resolução dos desafios propostos e servir como um portfólio prático da minha base em lógica e desenvolvimento web.

---

## 🎯 Objetivos de Aprendizagem
* Dominar a sintaxe moderna do JavaScript (ES6+).
* Compreender e manipular o DOM (Document Object Model) para criar páginas dinâmicas e interativas.
* Desenvolver lógica de programação sólida com estruturas condicionais e de repetição.
* Entender o funcionamento do Node.js como ambiente de execução no lado do servidor.

---

## 📂 Estrutura do Repositório

O projeto está organizado por módulos, seguindo a cronologia do curso:

*   📂 **`modulo-a/`**: Primeiros passos, história da linguagem e configuração do ambiente (VS Code e Node.js).
*   📂 **`modulo-b/`**: Tratamento de dados, variáveis, tipos primitivos e operadores (aritméticos, atribuição, relacionais e lógicos).
*   📂 **`modulo-c/`**: Entendendo o DOM, eventos e introdução à interatividade web.
*   📂 **`modulo-d/`**: Estruturas condicionais (`if/else`, `switch/case`) e exercícios práticos (ex: verificador de hora e idade).
*   📂 **`modulo-e/`**: Estruturas de repetição (`while`, `do while`, `for`) e desafios de contagem/tabuada.
*   📂 **`modulo-f/`**: Variáveis compostas (Arrays), Funções, Escopo e introdução a Objetos.
